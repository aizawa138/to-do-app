# to-do-app
A simple ToDo application with a timer created with HTML, CSS and JavaScript.

# Description
The user enters the name, description and the time to finish the task. When the "Add task" button is pressed, a task is added to the ToDo list with a timer telling you the remaining time to finish the task. The tasks will be lined in order of remaining time, making it useful for users.

# Tech stacks
- HTML
- CSS
- Vanilla JavaScript
